Desarrollo de aplicación web para **Toulouse Lautrec**, enfocada en la gestión académica y administración de información.

Se implementaron módulos de registro de usuarios, gestión de cursos y manejo de datos mediante estructuras tabulares, junto con la mejora de la interfaz adaptada a una nueva identidad visual.

**Impacto:** Mejora en la organización de datos, usabilidad del sistema y coherencia visual de la plataforma.

**Stack:** PHP, JavaScript, MySQL, HTML, CSS
