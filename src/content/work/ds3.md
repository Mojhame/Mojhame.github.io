## Sobre el Proyecto

Realicé el **rediseño frontend completo** para **DS3 Comunicaciones**, una empresa de telecomunicaciones que requería modernizar su presencia digital. El objetivo fue crear una interfaz atractiva, funcional y completamente responsiva que reflejara la calidad y profesionalismo de sus servicios.

## Trabajo Realizado

### Diseño UI/UX
- Rediseñé la estructura visual del sitio desde cero
- Implementé una paleta de colores moderna y alineada con la identidad de la marca
- Optimicé la jerarquía visual para mejorar la navegación
- Diseñé layouts responsivos que se adaptan a dispositivos móviles, tablets y desktop

### Desarrollo Frontend
- **HTML5 semántico** - Estructura limpia y accesible
- **CSS3 moderno** - Utilicé Flexbox y Grid para layouts flexibles
    - Sistema de diseño modular
    - Animaciones y transiciones suaves
    - Estilos optimizados para rendimiento
- **JavaScript vanilla** - Interactividad sin dependencias externas
    - Menú hamburguesa para móviles
    - Carrusel de servicios
    - Validación de formularios en tiempo real

## Funcionalidades Implementadas

| Módulo | Descripción |
|--------|-------------|
| **Header/Navbar** | Menú de navegación responsivo con efecto sticky |
| **Hero Section** | Banner principal con llamado a la acción |
| **Servicios** | Sección de servicios con tarjetas interactivas |
| **Formulario de Contacto** | Validación frontend con feedback visual |
| **Mapa Interactivo** | Integración de ubicación de la empresa |
| **Footer** | Información de contacto, enlaces rápidos y redes sociales |

## Mejoras Implementadas

- **Responsive Design** - Experiencia óptima en todos los dispositivos
- **Optimización de rendimiento** - Código limpio y carga rápida
- **Accesibilidad** - Mejores prácticas para usuarios con discapacidades
- **SEO on-page** - Estructura semántica para mejor posicionamiento
- **Cross-browser compatibility** - Compatible con navegadores modernos

## Tecnologías Utilizadas
- HTML5 → Estructura semántica
- CSS3 → Estilos, Flexbox, Grid, Animaciones
- JavaScript → Interactividad y lógica frontend
- Git → Control de versiones
- GitHub → Repositorio y despliegue



## Resultados

- Interfaz moderna y profesional
- Navegación intuitiva en todos los dispositivos
- Aumento de la retención de usuarios
- Código mantenible y escalable
- Tiempo de carga optimizado
