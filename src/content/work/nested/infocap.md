## Sobre el Proyecto

Desarrollé una aplicación web integral para **Infocap Perú** como proyecto freelance, diseñada para optimizar la gestión académica y administrativa de la institución. El sistema permite la administración eficiente de alumnos, cursos y certificaciones, con un enfoque especial en la seguridad y accesibilidad de la información.

## Características Principales


> Sistema de Autenticación

- **Login y Registro de usuarios** con diferentes niveles de acceso
- **Roles diferenciados**: Vista de administrador y vista de alumno
- Sistema de sesiones seguro con Laravel Sanctum

> Panel Administrativo

- **Gestión completa de tablas**: Alumnos, Cursos y Certificados
- CRUD completo para cada módulo
- Panel de control con métricas y estadísticas

> Portal del Alumno
- Visualización personalizada de datos académicos
- Consulta de cursos inscritos
- Descarga de certificados digitales

> Validación con QR
- **Generación dinámica de códigos QR** para cada alumno
- Página pública que permite validar identidad mediante DNI
- Acceso seguro a datos personales del alumno tras validación
- Verificación de documentos oficiales de Infocap

## Tecnologías Utilizadas

> Frontend
- **React.js** - Biblioteca para la interfaz de usuario
- **JavaScript** - Lógica de componentes y estado
- **TailwindCSS** - Estilos y diseño responsivo

> Backend
- **PHP / Laravel** - Framework para API RESTful
- **MySQL** - Base de datos relacional
- **Laravel Sanctum** - Autenticación API

> Herramientas
- **Git & GitHub** - Control de versiones y colaboración
- **Composer** - Gestión de dependencias PHP
- **npm** - Gestión de paquetes JavaScript

### Metodología de Trabajo

Este proyecto fue desarrollado siguiendo buenas prácticas de ingeniería de software:
- Control de versiones con **Git**, utilizando ramas principales (main) y commits descriptivos
- Arquitectura MVC con Laravel
- Componentes reutilizables en React
- API RESTful para comunicación frontend-backend
- Seguridad implementada en rutas y endpoints

### Resultados

La aplicación ha permitido a Infocap Perú:
- Digitalizar por completo la gestión académica
- Reducir tiempos administrativos en un 70%
- Ofrecer a los alumnos acceso 24/7 a su información académica
- Validar certificados de manera rápida y segura mediante QR
- Mantener un registro histórico de todos los alumnos y sus avances
